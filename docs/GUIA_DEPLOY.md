# Guia de Deploy

## Render.com

1. Crie conta em https://render.com
2. Conecte este repositório GitHub
3. Configure variáveis de ambiente
4. Deploy automático!

## Variáveis de Ambiente

| Key | Value |
|-----|-------|
| NODE_ENV | production |
| PORT | 10000 |
| ICECAST_HOST | localhost |
| ICECAST_PORT | 8000 |
| ICECAST_MOUNT | /stream |
| ICECAST_PASSWORD | sua_senha |
