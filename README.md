# 🎙️ Rádio Amigos do Seu Zé

Rádio comunitária de streaming de Pontos de Umbanda.

## 🚀 Deploy no Render.com

1. Crie conta em [render.com](https://render.com)
2. Conecte este repositório GitHub
3. Configure as variáveis de ambiente
4. Deploy automático!

## 🌐 URL Pública

```
https://radio-amigos-seu-ze.onrender.com
```

## 📁 Estrutura

- `backend/` - Servidor Node.js + API
- `frontend/` - Site da rádio (HTML/CSS/JS)
- `scripts/` - Scripts de deploy

## 🎵 Streaming

Configure o Mixxx para apontar para o servidor Icecast.
